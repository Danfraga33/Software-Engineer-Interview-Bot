### Software Engineer Chatbot

1. Languages:
   1. Backend: Python
   2. Frontend: React
2. API's:
   1. OPENAI - Whisper
   2. ElevenLabs

Directions

1. Clone
2. cd frontend/
3. npm i
4. cd ..
5. cd backend/
6. Get your own OPEN_AI_ORG, OPEN_AI_KEY and ELEVEN_LABS_API_KEY keys. Store them in .env
7. Run both sides of application, on seperate terminals.
